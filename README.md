# RevUnit

RevUnit take home assignment. A simple RESTful service for creating and fetching customers.

## Set up

1. Install packages

```bash
npm i
```

2. Install husky to set up pre commit hooks.

```bash
npm run prep-husky
```

3. Run dev

```bash
npm run start:watch
```

4. Build prod

```bash
npm run build
```
5. Run prod
```bash
npm run start
```
